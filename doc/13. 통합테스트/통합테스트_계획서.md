## 4. 주요 API 빠른 참조

### Backend API (Spring Boot :9000)
```
POST   /api/auth/login                         — 로그인 (세션 발급)
POST   /api/auth/logout                        — 로그아웃 (세션 무효화)
GET    /api/auth/check                         — 세션 확인 (세션 없으면 401)
GET    /api/auth/check-username?username=XXX   — 아이디 중복 확인
POST   /api/auth/signup                        — 회원가입
POST   /api/auth/find-id                       — 아이디 찾기 (마스킹)
POST   /api/auth/verify-reset                  — 비밀번호 재설정 본인 확인
GET    /api/dashboard/intersections            — 교차로 목록 (위경도 포함)
GET    /api/dashboard/weather                  — 날씨 정보
GET    /api/dashboard/today-violations         — 오늘 단속 집계
GET    /api/dashboard/road-congestion[?direction=UP/DOWN] — 도로 혼잡도
GET    /api/dashboard/traffic-summary          — 교통량 요약 (7일)
GET    /api/dashboard/cctv-status              — CCTV 현황
GET    /api/cctv/list                          — CCTV 목록 (is_active=true)
GET    /api/enforcement/violations[?page&size&plateNumber&violationType&status]
PUT    /api/enforcement/violations/{id}        — 위반 정보 수정
PUT    /api/enforcement/violations/{id}/status — 상태 변경 (승인/반려 — 한국어)
POST   /api/enforcement/webhook                — Webhook 수신 (세션 불필요)
```

GET    /health                                 — 헬스체크

GET    /api/v1/stream/violations[?limit=20]    — 과속 위반 캐시 목록
